# Guía de Imágenes Móviles para el Hero (Inicio)

Para lograr un rendimiento óptimo de tu página, un tiempo de carga rápido, y evitar que Google te penalice, tu archivo `portada_mobile.jpg` deberá cumplir de preferencia estas características:

## Dimensiones Recomendadas
- **Resolución Ideal:** `1080 x 1920 píxeles` (Formato retrato clásico de teléfono, 9:16).
- *Alternativa rápida:* `750 x 1334 píxeles`. 

## Peso y Formato
- **Peso Máximo Sugerido:** Entre `150 KB` y `350 KB`. ¡Trata de no pasar de 500 KB bajo ninguna circunstancia para asegurar una carga instantánea en conexiones 4G/3G!
- **Formato:** `.jpg` o `.webp`. (Si tienes un archivo .png y pesa mucho, expórtalo como JPG en calidad 70-80%).

## Encuadre y Composición
- Mantiene el elemento central (tu Osonauta / cohete / elemento principal) justo al medio o un poco por encima del centro `(bg-[center_top_20%])`.  
- Deja los bordes laterales y las esquinas con aire o fondo vacío; esta zona suele ser recortada ligeramente por pantallas más delgadas o más anchas.

## Tiempos de Carga
Actualmente el código está diseñado de manera que **los teléfonos solo descargan `portada_mobile.jpg`** y el escritorio solo descarga la versión original, ahorrando muchos megabytes en los datos de los usuarios.
